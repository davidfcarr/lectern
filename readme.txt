Lectern is a Toastmasters-friendly theme based on underscores.

== Description ==

While Lectern can be used for any purpose, it was designed as an easy-to-install theme for Toastmasters clubs and districts wanting to set up a site that uses WordPress but conforms to the organization's branding guidelines. It makes a good companion to the RSVPMaker for Toastmasters plugin.

== Installation ==
	
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
4. Toastmasters clubs: visit the Appearance > Toastmasters Branding screen to retrieve branded materials approved by Toastmasters International to be used in the site header.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

* Lectern works well with RSVPMaker and RSVPMaker for Toastmasters.
* Lectern includes support for Infinite Scroll in Jetpack.
